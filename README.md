# AutoSUID

AutoSUID application is the Open-Source project, the main idea of which is to automate harvesting the SUID executable files and to find a way for further escalating the privileges. The 100% automated.

Usage example:
--------------
```
chmod +x AutoSUID.sh
./AutoSUID.sh
```
or
```
bash ./AutoSUID.sh
```
